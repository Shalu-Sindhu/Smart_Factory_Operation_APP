# AI-Powered Smart Factory Operation Application

An AI-powered Smart Factory Operations Management System that enables real-time production monitoring, machine health analysis, worker management, AI-driven alerts, and operational analytics through an intuitive web dashboard.

## Overview

The AI-Powered Smart Factory Operation Application is designed to digitalize manufacturing operations by providing a centralized platform for factory monitoring and management. The application helps administrators, supervisors, and workers improve productivity, enhance workplace safety, and make data-driven decisions using AI-powered insights.
